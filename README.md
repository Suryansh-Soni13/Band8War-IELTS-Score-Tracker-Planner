## **Features**

- **User Authentication:** Simple login system using LocalStorage, no backend required.  
- **Score Management:** Add, view, and track scores for each IELTS module.  
- **20-Day Study Plan:** Daily tasks covering all modules, helping students stay organized and achieve their target score.  
- **Progress Analytics:** Interactive charts using Chart.js display performance trends, strengths, and improvement areas.  
- **Responsive Design:** Built with Tailwind CSS, fully functional on desktop and mobile devices.  

---

## **Technology Stack**

- **Frontend:** HTML, Tailwind CSS, JavaScript  
- **Data Storage:** LocalStorage for persistent user data  
- **Charts & Analytics:** Chart.js  

---

## **Usage**

1. Open the `index.html` file in a web browser.  
2. Enter a username to log in or register.  
3. Add IELTS scores for each module.  
4. Follow the 20-day study plan tasks.  
5. Track progress through interactive charts on the dashboard.
